Information on Bootstrap


Bootstrap is a toolkit designed for use with HTML5, that allows for styling using html classes, rather than large css style sheets. The base of the kit is derived from Sass and Javascript, running on a responsive grid-12 system, and has a massive database of built-in components to use.

One of the simplest components to use is the Button, as displayed below. You can disabled them, change their colour and text, their alignment and so much more!

Another commonly used component, is the Alert Box, which has specialized colour contrast settings and is used to notify the user of the response to action, such as it being completed, failed, an error occuring, or something else! Here is an example of a "danger" alert below.

This is an example of an Alert Box!

One of the components that takes advantage of the built in Javascript in Bootstrap is the Modal. It is essentially a pop-up that can include text, buttons, dropdowns and other components of bootstrap, but appears to be elevated off the page and can fade the background behind it to draw the focus of the viewer. Click the button below to view an example of a modal with some customized buttons and text.

The last component I will showcase here is the card. it is a simple block element that can house text, pictures, or other components. They are useful for creating isolated "blocks" on your page that can easily group together pictures, text and components in a neat looking way.
